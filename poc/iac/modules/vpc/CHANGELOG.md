## [1.0.0] - 2024-XX-XX

### Added
- Initial release of the VPC module
- Support for creating a VPC with configurable CIDR blocks
- Public and private subnet creation across multiple availability zones
- Internet Gateway configuration
- NAT Gateway configuration for private subnets
- Route table configuration
- Security group management
- Network ACL configuration
- VPC endpoint configuration
- Tagging support
